Folder Structure

FolderStructure এর মাধ্যমে componenet গুলাকে store করে রাখা হয় ।
যেমন মাঝারি ধরনের project হলে components নামের একটা folder নিয়ে সেটার মধ্যে সব গুলা componenet store করে রাখে
অনেকে প্রতিটা page এর জন্য আলাদা আলাদা FolderStructure তৈরি করে ,
আর SPA এর ক্ষেত্রে প্রতিটা viwe এর জন্য আলাদা আলাদা folder বানায়
আবার কেউ প্রতিটা componenet এর জন্য আলাদা আলাদা folder করে তবে কিছু shared বা common folder করে ।
এক এক জনে এক এক ভাবে করে ।

FolderStructure তৈরি করতে হলে প্রথমে src এর মধ্যে একটা Components নামের একটা folder তরি করতে হবে তারপর তার ভেতর একটা folder নিতে হবে component এর নামে ।
তারপর componenet কে রাখার জন্য একটা js ফাইল এবং তাকে style দেয়ার জন্য একটা css ফাইল নিতে হবে .
