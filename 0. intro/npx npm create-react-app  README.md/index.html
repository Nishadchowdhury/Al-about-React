What is npm

____ npm হল node package manager ____
যেমন বিরিয়ানি কিনতে গেলে , আমি যদি বলি ১ প্যাকেট বিরিয়ানি দেন তখন দোকানি যেমন , সকল উপাদান প্রয়োজন মত এক করে আমাকে দেয় ।
ঠিক একই ভাবে npm একটা package manager যেটা আলাদা আলাদা অনেক গুলা জিনিস কে এক সাথ করে আমাদের react project এর জন্য install করে দেয়

What is npx

এটার কাজ হল node package executation
কিছু বছর আগে npx এর কাজটা ২ ভাগে করা হতো , প্রথমে create-react-app টাকে install করতাম তারপর cerate-react-app এর নাম দিয়ে একটা project তৈরি করে কাজটা করতে হতো ।
npx এর সাহায্যে সে আর চেক করবে না install আছে কিনা , যদি না থাকে সে install করবে আর থাকলে সে সরাসরি একটা project তৈরি করে ফেলে ।
অর্থাৎ সে install এবং execute এর কাজটা এক বারেই করে ফেলে ।

What is create-react-app

এটা এক ধরনের CLI
Command line Interface
অর্থাৎ এটার কাজ হল react এর application তৈরি করার একটা command line Interface .
তারপর
আগে ধরে ধরে এগুলাকে setup করে লাগতো , যেমন app.js এর মধ্যে একটা defult componenet থাকে সেটাকে আগে ধরে ধরে তৈরি করা লাগতো ,
তারপর index.js এটাকে বলে কোন একটা application এর entry point অর্থাৎ কোথা থেকে application টা পুরাপুরি শুরু হচ্ছে ।
এবং প্রয়োজনীয় file গুলাকে setup , import করত এবং full application কে root element এর মধ্যে সেট করত
অর্থাৎ এই full জিনিস টাকে ready করে html এর মধ্যে বসানর কাজটা এই create-react-app করে দিচ্ছে শুধু মাত্র একটা command এর মাধ্যমে ।
